## How It Works

<Steps>
  <Step title="Enter Repository URL">
    Paste any GitHub repository URL or use the `owner/repo` format
  </Step>
  <Step title="Auto-Detection">
    i18n Doctor scans the repository tree and auto-detects translation files in common patterns
  </Step>
  <Step title="Analysis">
    Compares all target locales against the source locale to identify missing keys, untranslated strings, and orphan keys
  </Step>
  <Step title="Health Report">
    View a comprehensive dashboard with per-locale coverage stats and expandable issue lists
  </Step>
  <Step title="One-Click Fix">
    Use Lingo.dev to translate all gaps and download fixed files or create a GitHub PR
  </Step>
</Steps>
